M.S. Thesis  ·  Data Engineering & Machine Learning

Arlo Hotels Demand Forecasting System

Built a production-ready demand forecasting pipeline using two years of proprietary reservations and revenue data from Arlo Hotels. A Python ETL process ingests and cleans PMS exports; a LightGBM model trained on engineered features — day-of-week, lead time, seasonality, local event calendar — achieves R² 0.97 with roughly 7% MAPE across properties. A Streamlit dashboard gives revenue managers live 30/60/90-day occupancy forecasts without touching the underlying model.
